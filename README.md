# a2d

#### 介绍
将AppImage程序包转换成Deb软件包。

#### 安装教程

1.  git clone https://gitee.com/qygw/a2d
2.  cd a2d/src
3.  mkdir build &&cd build
4.  qmake ..
5.  make

#### 依赖

1.  qt5那一套
2.  fakeroot
